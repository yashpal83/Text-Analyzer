This is a simple text-analyzer. You can use it for some basic text analyzation, Like:

  1. Removing Punctuation Marks

    2. Using this operation you can remove all the punctuation marks from the text.

  2. Removing Numbers

    3. Using this operation you can remove all the Numbers from the text.

  3. Capitalize First

    4. You can use this operation for Capitalizing the first character of the text.

  4. UPPERCASE THE TEXT

    5. This operation will convert your whole text to UPPERCASE.

  5. lowercase the text

    6. This operation will convert your whole text to lowercase.

  6. Extra Spaces Remover

    7. If you have extra spaces in the text, you can use this operation for clearing the extra spaces from the text.

  7. Spaces Remover

    8. This operation will remove all the spaces from the text.

  8. Extra Lines Remover

    9. This operation will be helpful when you want to remove the extra lines from the text.

  9. Count Charaters

    10. This operation will count all the characters present in the text.

Note: You can use multiple operations at the same time.
